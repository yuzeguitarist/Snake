# 🐍 极简贪吃蛇游戏

一个采用苹果风格设计的极简贪吃蛇游戏，支持自动主题切换和多设备适配。

## ✨ 特性

- 🎨 **苹果风格设计** - 简洁、优雅、美观的界面
- 🌓 **自动主题切换** - 根据系统偏好自动切换亮色/暗色主题
- 📱 **多设备适配** - 完美支持桌面端和移动端
- ⌨️ **灵活操控** - 支持键盘方向键和触摸滑动操作
- 💾 **记录保存** - 自动保存最高分记录
- 🎯 **流畅体验** - 优化的游戏逻辑和动画效果

## 🎮 操作说明

### 桌面端
- **方向键** / **WASD** - 控制蛇的移动方向
- **空格键** - 开始/暂停游戏

### 移动端
- **滑动** - 控制蛇的移动方向
- **点击开始按钮** - 开始游戏

## 🚀 快速开始

直接在浏览器中打开 `index.html` 文件即可开始游戏。

```bash
# 如果使用本地服务器
python -m http.server 8000
# 或
npx serve
```

然后访问 `http://localhost:8000`

## 🎨 设计理念

本项目采用苹果公司的设计语言：

- **极简主义** - 删除所有不必要的元素，只保留核心功能
- **系统字体** - 使用 San Francisco 字体系统
- **柔和圆角** - 所有元素采用圆角设计
- **细腻阴影** - 精心调配的阴影层次
- **流畅动画** - 所有交互都有平滑的过渡效果
- **响应式设计** - 适配各种屏幕尺寸

## 🌈 主题系统

游戏支持亮色和暗色两种主题：

- **自动切换** - 根据系统偏好自动选择主题
- **手动切换** - 点击右上角按钮手动切换
- **记忆功能** - 保存用户的主题偏好

## 🔧 技术栈

- **纯原生 JavaScript** - 无依赖，纯净的实现
- **Canvas API** - 流畅的游戏渲染
- **CSS Variables** - 灵活的主题系统
- **LocalStorage** - 本地数据存储
